Задание

Сделать форк данного проекта на github.com, склонировать форк на свой компьютер, внеси изменения — добавить себя в список контрибьютеров в разметке, закоммить изменения в форк, создать из форка pull request в совновной репозиторий.

Техническое задание

  1. На рабочем компьютере установить и настроить git: инструкция https://htmlacademy.ru/blog/useful/git/git-console
  2. Зарегистрироваться на github.com
  3. Сделать форк этого репозитория на github — https://github.com/igosheva/git-task
  4. Склонировать форк на свой компьютер
  5. Внеси изменения – добавить себя в список контрибьютеров в разметке html
  6. Закоммитить изменения в форк
  7. Создать из форка pull request в совновной репозиторий, из котрого был создан форк
